# Privacy Policy / 隱私權政策

**Last updated / 最後更新：2025-01-17**

---

## English

### What data does this extension collect?

**Lee-Su-Threads does not collect any personal data.**

The extension only:

- Intercepts API responses from threads.com to extract publicly available profile information (location, join date) of post authors
- Stores this data locally in your browser's storage
- Does not transmit any data to external servers

### Data Storage

- All data is stored locally on your device using your browser's built-in storage API
- Cached data automatically expires after 24 hours
- You can clear all stored data at any time through the extension popup

### Permissions Used

- **storage**: To cache profile data locally for better performance
- **host_permissions (threads.com)**: To run the extension on Threads website

### Third-Party Services

This extension does not use any third-party analytics, tracking, or data collection services.

### Contact

If you have any questions about this privacy policy, please open an issue on our GitHub repository.

## 繁體中文

### 本擴充功能收集哪些資料？

**Lee-Su-Threads 不收集任何個人資料。**

本擴充功能僅：

- 攔截 threads.com 的 API 回應，擷取貼文作者的公開個人資訊（地點、加入時間）
- 將資料儲存在您瀏覽器的本機儲存空間
- 不會將任何資料傳送到外部伺服器

### 資料儲存

- 所有資料都使用瀏覽器內建的儲存 API 儲存在您的裝置上
- 快取資料會在 24 小時後自動過期
- 您可以隨時透過擴充功能彈出視窗清除所有儲存的資料

### 使用的權限

- **storage**：用於在本機快取個人資料以提升效能
- **host_permissions (threads.com)**：用於在 Threads 網站上執行擴充功能

### 第三方服務

本擴充功能不使用任何第三方分析、追蹤或資料收集服務。

### 聯絡方式

如果您對本隱私權政策有任何疑問，請在我們的 GitHub 儲存庫開啟 issue。
